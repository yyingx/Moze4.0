# Moze 4.0 账单转换工具

用于将支付宝、微信和云闪付账单转换为 Moze 4.0 可导入格式，并提供账单备注同步、云闪付 OCR 和差旅报销生成工具。

- 商家字典使用规范
- OCR、备注同步和报销工作流
- 版本管理与提交要求

## 隐私说明

请勿提交支付宝、微信、云闪付源账单，Moze 导出文件、截图、日志及其他包含个人财务信息的文件。
