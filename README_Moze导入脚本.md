# Moze 导入脚本 v11.66

将支付宝/微信账单自动转换为 Moze 4.0 导入格式的 Python 脚本。

## 功能特性

- ✅ 支持支付宝 CSV 和微信 XLSX 账单
- ✅ 自动识别淘宝订单
- ✅ 备注驱动分类（如 `探索claude` → 子类=探索，描述=claude）
- ✅ 食材/正餐智能分类（根据关键词和时间推导）
- ✅ 转账自动生成转入/转出记录

## 快速开始

### 1. 准备文件
```
你的目录/
├── Moze4_0_Import_v11_66.py    # 脚本
├── Moze Dict.xlsx               # 规则文件（必需）
└── 支付宝账单.csv               # 待导入账单
```

### 2. 修改配置
打开脚本，修改以下内容：

```python
# 转账对方识别
CONFIG = {
    'TRANSFER_TARGET_1': '张三',           # 改成你的转账对象
    'TRANSFER_TARGET_2': '工商银行(1234)', # 改成你的银行
    ...
}

# 银行卡号映射
STANDARDIZE_ACCOUNTS = {
    r'.*1234.*': '工商银行',  # 卡号含1234 → 工商银行
    ...
}
```

### 3. 运行
```bash
python Moze4_0_Import_v11_66.py
```

## 备注语法

在支付宝/微信备注中使用关键词，可自动分类：

| 备注示例 | 结果 |
|---------|------|
| `探索claude` | 子类=探索，描述=claude |
| `日用洗衣液` | 子类=日常用品，描述=洗衣液 |
| `正餐麦当劳` | 根据时间→早餐/午餐/晚餐，描述=麦当劳 |
| `借出张三` | 子类=借出，对象=张三 |

## 自动识别

脚本会根据商品名/商家名自动识别：

| 关键词 | 分类 | 名称 |
|-------|------|------|
| 苹果、香蕉、草莓... | 饮料水果 | 水果 |
| 可乐、奶茶、咖啡... | 饮料水果 | 饮料 |
| 排骨、五花肉... | 食材 | 猪肉 |
| 三黄鸡、鸡腿... | 食材 | 禽肉 |
| 抽纸、洗衣液... | 日常用品 | - |

## 常见问题

**Q: 提示找不到 Moze Dict.xlsx？**
A: 将规则文件放在脚本同目录，或修改 `POSSIBLE_PATHS`。

**Q: 转账没有自动识别？**
A: 检查 `CONFIG['TRANSFER_TARGET_x']` 是否与账单中的"交易对方"完全一致。

**Q: 如何添加新的食材关键词？**
A: 在 `DATA_SOURCE` 对应类别中添加，如 `'PORK': ["猪肉", "排骨", "你的关键词"]`。
